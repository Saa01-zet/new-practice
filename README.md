Практика Git

Данный репозиторий создан для выполнения практических заданий по работе с Git.

Инструкция по развёртыванию
1. Собрать проект
2. Запустить тесты
3. Развернуть на сервере

Структура проекта
- docs/ - документация проекта
- src/ - исходный код
- backups/ - резервные копии
- newstuff/ - новые функции (ветка newbranch)
- oldstuff/ - стабильная версия (ветка master)

Ветки
- master - основная ветка (стабильная версия)
- newbranch - ветка для разработки новых функций
